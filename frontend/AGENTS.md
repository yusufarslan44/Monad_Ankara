# Frontend AI Rules

## Editable scope

AI agents may edit files in this directory when the task is a frontend task.

## Restrictions

1. Do not edit `../backend/`.
2. Do not edit `../contract/`.
3. Other repository folders may be read for context, but they are read-only unless the user explicitly approves broader changes.
4. Keep frontend commits isolated to reduce merge conflicts.
5. For Monad frontend work, `https://skills.devnads.com/` may be used as a dynamic skill reference, but only `frontend/` remains editable.
6. Never commit real frontend env values; only commit example env files with placeholder values.
